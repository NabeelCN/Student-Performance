Student Performance Classification
Project Overview

This project focuses on predicting student performance using Machine Learning classification techniques. The objective is to analyze student-related factors such as study habits, attendance, and academic information to classify whether a student is likely to perform well or poorly. The project helps educational institutions identify at-risk students and provide timely support.

Objectives
Perform data cleaning and preprocessing
Conduct Exploratory Data Analysis (EDA)
Analyze factors affecting student performance
Build and evaluate classification models
Predict student performance categories accurately
